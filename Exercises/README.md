## Run Pytest Coverage Plugin
```bash
(PythonWorkout) michael@ubuntu:~/PycharmProjects/PythonWorkout/Exercises$ pip install pytest-cov
(PythonWorkout) michael@ubuntu:~/PycharmProjects/PythonWorkout/Exercises$ pytest --cov Exercises
(PythonWorkout) michael@ubuntu:~/PycharmProjects/PythonWorkout/Exercises$ coverage html
```
